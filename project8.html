Pyuk

Pyuk is a 2D physics platformer game made in 48 hours for Global Game Jam 2024.

Date

Jan 2024

Role

Programmer

Project Type

Game Jam - Team

Link

Play here!

Engine

Unity

Language

C#

My work

  • Modular code implementing consumption and discharge of the critters via Unity's Action-Event systems that allow easy integration into other developers' code.

  • Spike gameplay element that causes the player to discharge and if already has nothing to discharge, die.

  • Breakable walls that can be destroyed depending on the hit force.

  • A filling mechanic that allows the player to fill a pit on the terrain with discharge and therefore enable passage by rolling over the raised elevation.

  • Integration of some sound effects.

  • Integration of systems to the UI.

  • Some level design.

Outcomes

During this game jam I learned a lot about writing code that is easy to integrate into other systems of the game, working closely with my teammates.

As it was a Global Game Jam, I had the pleasure of meeting and socializing with other developers at the jam site, even finding a new teammate.

Teaming up with two seasoned Unity developers, I was dedicated to learning from their experiences. Had a lot of long talks regarding best practices, code architecture and design.
